Installation:

Download MongoDB: https://www.mongodb.com/try/download/community

Install it after downloading without changing any path.

After Installation go to your C: Drive and create a folder with name data and inside this folder create another folder db. keep both of them empty.

Now Go to your C:/programfiles->Mongodb->server->4.0 or whatever version you have installed->bin copy path
open cmd and go to that copied path
write mongod and hit enter


Running:

Now extract you Project.zip folder in C:/users/public or your PC name/

Run server-win.exe

Now open your browser and enter localhost:4444
